# UX/UI Review — czerwiec 2026

## Problem

Interfejs jest wyłącznie po polsku, ale zawierał wyciekające angielskie ciągi: "Snapshots",
"Total compensation", "Equity", "Discount", "liquidity event", "double-trigger", "Vested",
"FMV", "Dashboard".

## Zmiany

Przetłumaczono wszystkie ciągi znaków na język polski. Poniżej glosariusz terminów
finansowych/technicznych, które pozostają w UI ze względu na powszechne użycie lub brak
polskiego odpowiednika.

## Glosariusz zatrzymanych terminów

| Termin | Pełna nazwa                   | Objaśnienie                                                                                                                               |
| ------ | ----------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| 409A   | Wycena 409A                   | Niezależna wycena wartości godziwej akcji zwykłych spółki prywatnej wymagana przez przepisy podatkowe USA (IRC §409A).                    |
| ESOP   | Employee Stock Ownership Plan | Program akcji pracowniczych — forma wynagrodzenia w postaci udziałów lub opcji na akcje.                                                  |
| FX     | Foreign Exchange              | Kurs wymiany walut. Używany przy przeliczaniu wartości grantów denominowanych w USD na PLN.                                               |
| IPO    | Initial Public Offering       | Pierwsza oferta publiczna — wejście spółki na giełdę. Często wyzwalacz dla nabywania udziałów z podwójnym wyzwalaczem.                    |
| MoM    | Month over Month              | Zmiana miesiąc do miesiąca — różnica względem wartości sprzed ~1 miesiąca.                                                                |
| PLN    | Polski złoty                  | Kod waluty ISO 4217 dla polskiego złotego.                                                                                                |
| RSU    | Restricted Stock Unit         | Jednostka akcji z ograniczeniami — forma wynagrodzenia w postaci akcji, które są przyznawane po spełnieniu warunków nabywania (vesting).  |
| SWR    | Safe Withdrawal Rate          | Bezpieczna stopa wypłat — maksymalny roczny procent portfela, który można wypłacić bez ryzyka wyczerpania środków przez okres emerytalny. |
| YoY    | Year over Year                | Zmiana rok do roku — różnica względem wartości sprzed ~12 miesięcy.                                                                       |
